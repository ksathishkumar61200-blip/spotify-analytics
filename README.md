# spotify-analytics
🎧 Spotify India Tracks + Analytics Dashboard

📌 About
This project collects Indian songs from Spotify using its API, stores them in a MySQL database, and visualizes insights using a Power BI dashboard.
It focuses on songs from the year 2024.

✨ Features
Fetch songs using Spotify API
Store data in MySQL
Export data to CSV

📊 Interactive dashboard for analysis
🛠️ Technologies
Python
Spotipy
Pandas
MySQL
Power BI

▶️ Run Project
python main.py
📊 Dashboard Preview

📊 Dashboard Insights
🎵 300 Total Tracks collected
🎤 188 Artists analyzed
💿 206 Albums covered
⏱️ Average Duration: 4.11 mins
Visual Analysis Includes:
Top artists by number of tracks
Monthly track release trends
Album-wise track distribution
Duration analysis (bins)
Source playlist contribution
